SIDURIM React PDF v3

איך מריצים:
1. חלץ את הקובץ
2. פתח CMD בתיקייה
3. הרץ:
   npm install
   npm run dev

מה חדש ב-v3:
- בעמוד הבית מופיעות רק משימות עם חשיבות גבוהה.
- ברובליקת בנייה > בעלי מקצוע / ספקים והשוואות:
  אפשר להכניס נייד, מייל, כתובת, כספים וחוזים בתוך הכרטיס.
- עדיין כולל העלאת PDF בעבודה > הזמנות עבודה.
